# Capon Test Data — Откуда брать данные

## Источник данных

Все файлы данных — из **эталонного прототипа заказчика** (ArrayFire, CPU).

| Оригинал (прототип) | Локальная копия (тесты) |
|---------------------|------------------------|
| `Doc_Addition/Capon/capon_test/build/` | `modules/capon/tests/data/` |

**Тесты используют локальную копию** через `kDataDir` в `capon_test_helpers.hpp`.

## Файлы данных

| Файл | Размер | Формат | Описание | Используется |
|------|--------|--------|----------|:------------:|
| `x_data.txt` | 340 строк | float (1 на строку) | Координаты x антенных элементов (метры) | **ДА** |
| `y_data.txt` | 340 строк | float (1 на строку) | Координаты y антенных элементов (метры) | **ДА** |
| `signal_matlab.txt` | 341 строк | MATLAB complex (`re+imi`) | Сигнальная матрица [341 x 1000] complex | **ДА** |
| `z_values.txt` | 5476 строк | float (1 на строку) | Эталонные значения рельефа (4 блока x 37x37) | НЕТ* |
| `interf0_matlab.txt` | 341 строк | MATLAB complex | Матрица помех (из прототипа, `is_from_file=true`) | НЕТ |

*`z_values.txt` — эталон рельефа от прототипа (mu=1000, **без** деления на N). Пока не используется в GPU тестах (формула отличается: GPU делит на N). Может быть полезен для верификации формы рельефа.

## Как сгенерировать данные заново

1. Собрать прототип:
   ```bash
   cd Doc_Addition/Capon/capon_test/build
   cmake .. && make
   ```
   Требует: ArrayFire, AMD GPU (или Intel с OpenCL)

2. Запустить (интерактивный):
   ```bash
   ./capon_test
   # P=85, N=1000, typeCalc=4 (chol+inv), time_flag=1
   ```

3. Результат: `z_values.txt` в `build/`

4. Входные файлы (`x_data.txt`, `y_data.txt`, `signal_matlab.txt`) — **статические**, сгенерированы заранее в MATLAB.

## Физические параметры

```
f0 = 3918 МГц + 3.15 МГц = 3 921 150 000 Гц   (несущая частота)
c  = 299 792 458 м/с                             (скорость света)
P  = 85 каналов                                  (подмассив из 340)
N  = 1000 отсчётов
```

## Формат MATLAB complex

Каждая строка signal_matlab.txt — 1000 комплексных чисел через пробел:
```
-12.4903+15.0831i 6.1672+18.7393i -9.8028+30.5967i ...
```

Парсер: `ParseMatlabComplex()` в `capon_test_helpers.hpp`.

## Отличие формулы: Прототип vs GPU

| | Прототип (ArrayFire) | GPU (CaponProcessor) |
|---|---|---|
| Формула | R = Y*Y^H + mu*I | R = (1/N)*Y*Y^H + mu*I |
| mu (типичное) | 1000 | 1.0 |

Для сравнения **форм** рельефа (не абсолютных значений) — нормировка не влияет.
